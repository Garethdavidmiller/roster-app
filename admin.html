Admin Tips
Changing shifts
1️⃣Select a staff member and week at the top
2️⃣Tap a shift type button on any day to change it
💾Tap Save — unsaved changes are highlighted in amber
👆Swipe left / right on the week grid to move between weeks
Bulk changes
☑️Check multiple days, choose a type and time, then tap Apply to selected days
Annual leave
🏖️Use the Record Annual Leave section — rest days within the range are skipped automatically
⚠️A warning appears if leave would exceed the member's entitlement — you can still save
Override types
🔄Swap — use when a member swaps a shift with a colleague
✏️Make Rest Day — corrects a rostered working day to a rest day
