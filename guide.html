The Buttons

🏖️ AL
Your annual leave balance — Taken · Booked · Remaining · Entitlement. Tap "Record leave in Admin" to book leave.
📋 Huddle
Opens the latest daily briefing. Updated automatically each day. Tap to open in your PDF viewer.
💳 Pay
💷 = payday (every 4 weeks, Friday). ✂️ = expenses cut-off (Saturday before payday).

Install as an App

Android (Chrome)
Tap the browser menu (⋮)
→ Add to Home Screen
iPhone / iPad (Safari)
Tap the Share button (□↑)
→ Add to Home Screen
Push notifications — open Admin → scroll to Notifications → tap Subscribe. You'll then be notified when each day's Huddle is uploaded. iPhone: must use Safari with the app added to Home Screen first.

Works Offline

Your base roster is stored on your device after first load — the calendar always works without signal. Shift changes recorded by admin and the Huddle link need internet to load.

Marylebone Roster — Admin Guide

Chiltern Railways · MYB Station · Admin and self-service users
v6.x

Logging In

  1. Tap the 🔒 Admin button on the main screen
  2. Select your name from the dropdown
  3. Enter your surname as the password — lowercase, no spaces or punctuation
  4. Tap Log In — sessions last 30 days
Example: "D. Smith" → password is smith

Recording a Shift Change

  1. Select the staff member from the top dropdown
  2. Enter or navigate to the week containing the day
  3. Tick the checkbox beside the day(s) to select them
  4. Tap a shift type button to choose the type
  5. Enter Start and End time if the type requires one
  6. Add an optional note for context
  7. Tap Save Changes

Apply to Multiple Days

Tick several days in the week grid — the Apply to selected days bar appears. For a holiday spanning several days, use the Record Annual Leave card instead — pick a date range and tap Record.

Saved Changes

Scroll to the Saved Changes card. Filter by member and month using the dropdowns. Tap the bin icon to remove an entry. Entries tagged PDF upload were saved automatically by the weekly roster upload — you can still delete or edit them if needed.

Weekly Roster Upload (admin only)

  1. Open the Weekly Roster card in admin
  2. Select the roster type — CEA, CES, or Dispatcher
  3. Set the week-ending date (always the Saturday of that week)
  4. Select the PDF file from your device
  5. Tap Read Roster — AI reads the PDF (takes ~15 seconds)
  6. A per-person review appears — approve or skip each day's change
  7. Tap Apply X changes to save the approved changes
Note: The file must be a PDF. The week-ending date must be a Saturday. If the AI misreads a shift, use Skip on that day and record it manually via Change a Shift.
